# My Noctalia Rice

My personal Arch Linux + Niri + Noctalia setup.

## Features

- Niri
- Noctalia
- Alacritty
- GTK themes
- Wallpapers

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/my-noctalia-rice
cd my-noctalia-rice
./install.sh
```

## Restore configs

```bash
./restore.sh
```
